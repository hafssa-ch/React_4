# 📝 To-Do List Interactive

Une application de gestion de tâches moderne et intuitive développée avec **React**.

## ✨ Fonctionnalités

- ✅ Ajouter des tâches
- ✅ Marquer les tâches comme terminées
- ✅ Modifier des tâches (double-clic ou bouton ✏️)
- ✅ Supprimer des tâches individuelles
- ✅ Supprimer toutes les tâches
- ✅ Compteur de tâches restantes
- ✅ Sauvegarde automatique dans le navigateur (localStorage)
- ✅ Messages d'erreur pour les champs vides
- ✅ Interface moderne avec animations

## 🚀 Technologies utilisées

- React 18
- Hooks (useState, useEffect)
- CSS3 (animations, flexbox)
- LocalStorage API



## 📁 Structure du projet
````

src/

├── components/

│   ├── TodoForm.js      # Formulaire d'ajout
│   ├── TodoList.js      # Liste des tâches
│   └── TodoItem.js     # Élément individuel

├── App.js               # Composant principal

├── App.css              # Styles CSS

└── index.js             # Point d'entrée

````
